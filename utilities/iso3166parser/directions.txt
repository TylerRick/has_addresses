NOTE:

The ISO files have been modified from their original source (Debian). DO NOT RUN OR REPLACE ISO FILES unless critical

1. Download the latest copy of iso-codes from http://pdo.debian.net/unstable/misc/iso-codes
2. Verify country codes with http://userpage.chemie.fu-berlin.de/diverse/doc/ISO_3166.html
3. Get the latest calling codes from http://en.wikipedia.org/wiki/List_of_country_calling_codes.
3. Run the following command:

ruby parse.rb iso_3166-1.txt iso_3166-2.txt calling_codes.txt countries.yml regions.yml

4. Search for the string ??? in the regions sql file.  This indicates a problem with the countries (you're mapping a region to an unknown country).
